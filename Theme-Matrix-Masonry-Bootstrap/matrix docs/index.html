Ajax suport

Ajax code is located in js/matrix.js

By default all clicks on a elements is captured and processed, for external links the code returns true, for links that have ajax_right class the content is loaded on the right #ajax container with slide animation.

for links that doesn't have ajax_right class links are loaded in the #container div and replaces the whole page content except top menu.

Sliders

Matrix theme comes integrated with 3 sliders

  1. Nivo slider
  2. Camera slider
  3. Video slider

Panel styles

To use styles from demo template panel, for example colors you need to include the coressponding css file from nico_theme_editor/styles folder

Retina

Retina support is provided by http://retinajs.com/ all you need to do to add support for retina is to add the higher resolution images in the same folder as original and append @2x in the filename, ex for test.png you need to have test@2x.png for retina version, more info here http://retinajs.com/

Effects

For each effect type there is a coresponding html file, the basic difference between them is the effect_* class from #content container that defines what effect is used for items, the page also has js to control the animation at the bottom of the page, there you can find animation_delay and duration variables to control animation speed

Fonts

The theme uses Font Awesome for item icons and Open sans and Monserrat for text, all of them are located in fonts folder, additionally the html code also contains code to use open sans and monserrat from google cdn if desired, you can find the commened code in the head section
